Scilab Statistical Prediction Script

Domenico Caputo, Domenico Guardini

The script is realized to perform prevision on periodic historical series. It gives good results if the correlation of the data between a period and the adjacent ones is high enough. It is possible to predict also series with a linear trend, but the method does not work with trends of a greater degree.

With the script and the header is also given a file containing three months of electrical consumptions that took place in a company. The parameters into the header file are optimized to give the best result with this data. From within the Scilab environment it is possible to run the script and verify how it works just typing:

exec('/path/to/script/dom_method.sci');
Have fun!
